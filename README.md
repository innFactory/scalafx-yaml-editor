# scalafx-yaml-editor
This is a ScalaFX demo that contains an simple YAML editor with syntax highlighting and a search function.
<br />


<img src="screenshot.png" width="90%">

<br />
<br />

## Run Project
<br />

```
cd scalafx-yaml-editor
sbt run
```
<br />
